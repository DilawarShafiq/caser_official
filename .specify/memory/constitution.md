# CASER Project Constitution

## 1. Project Identity
- **Name:** CASER (Center of Advanced Survey and Economic Research) website.
- **Mission:** Building the Future of Evidence-Based Research in Pakistan.
- **Aesthetic:** Professional, authoritative, clean, stunning.
- **Visuals:**
    -   **Fonts:** Montserrat (Headings), Open Sans (Body).
    -   **Colors:** Primary (Deep Navy/Professional Blue), Accent (Gold).
    -   **UI:** Rounded buttons, minimalist line icons, grid layouts, whitespace.

## 2. Tech Stack
- **Framework:** Vite-React-TS.
- **Styling:** Tailwind CSS.
- **Animation:** Framer Motion.
- **Language:** TypeScript.

## 3. Workflow
- Follow Spec-Driven Development (SDD).
- Specs -> Plans -> Implementation.
